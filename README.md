このコードは、郵便番号と住所の一覧を日本郵便のサイトから取得するものです。

## 対象となる郵便番号データ

[files.py](files.py) に記載している郵便番号データが対象です。
全住所の郵便番号と、全事業所の郵便番号です。
毎月の追加分・削除分のデータは対象外です。

## コードの説明

- `python main.py` を実行すると、郵便番号と住所・事業所の一覧を取得して、メモリ内に保存します。
  `find`メソッドを用いて、郵便番号から住所・事業所を検索するサンプルコードが実行されます。
- `python load_and_save.py` を実行すると、郵便番号と住所・事業所の一覧を取得して、それぞれのCSVファイルに保存します。
  保存先は `data/` ディレクトリです。
